# Patent Drafting Skill

面向中国发明专利撰写的 skill，用于根据技术交底、权利要求草案、检索结果、公式和用户模板生成或修订专利文本。

## 适用场景

- 撰写权利要求、说明书、具体实施方式、实施例
- 将技术方案整理成专利表达
- 检查公式变量和步骤衔接后再生成文本
- 根据检索结果调整撰写边界

## 核心文件

- `SKILL.md`：撰写规则、流程和输出格式

## 使用示例

```text
请使用 patent-drafting-skill，根据以下技术交底撰写一组中国发明专利权利要求。
重点保证前序步骤产物被后续步骤使用，公式变量定义清楚。
```
