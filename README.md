# whhhhhua-skills

`whhhhhua-skills` 是一个个人 skill 仓库，用来集中存放可复用的 Codex / Claude 风格技能包。

后续如果继续新增 skill，建议每个 skill 都放在仓库根目录下的独立文件夹里，并保持统一结构，例如：

```text
skill-name/
├── SKILL.md
├── README.md
└── examples/
```
